# Curso de Sistemas Operativos - 2026

Repositorio dedicado a las prácticas, laboratorios y proyectos de la universidad. 

## 💻 Entorno de Desarrollo
* **SO:** Ubuntu 24.04 (VirtualBox)
* **Recursos:** 2 CPUs / 4GB RAM
* **Herramientas:** GCC, Git, VS Code

## 📂 Estructura del Proyecto
* `/Tareas`: Ejercicios semanales de clase.
* `/Laboratorios`: Proyectos de implementación de procesos e hilos.

## 🚀 Cómo compilar
Para compilar cualquier archivo en C dentro de este repo:
`gcc nombre_archivo.c -o ejecutable`
